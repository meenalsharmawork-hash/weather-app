A simple weather application that shows real-time weather data using WeatherAPI.

## 🚀 Features
- Search weather by city
- Shows temperature, condition, and local time
- Dynamic weather icons
- Responsive design

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- WeatherAPI
